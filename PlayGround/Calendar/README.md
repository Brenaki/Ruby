# Calendar

## How to use application?
1. Clone the repository:
```
git clone https://github.com/Brenaki/Ruby.git
```

2. Go to Calendar:
```
cd .\PlayGround\Calendar\
```

3. Install require:
```
gem install colorize
```

4. Run the application:
```
ruby app.rb
```
